Travis CI Account

Used in lessons: Continuous Integration

Travis CI provides infrastructure execute automated testng and deployment tasks. It is easily integrated with hosting services, like GitHub.

You will need an account at travis-ci.com for the continuous integration testing activity. Basic Travis accounts are free. When creating your account, you should link it to your GitHub account. Of course, this requires that you setup your GitHub account first.

Setting Up Your Travis CI Account

  1. Go to https://travis-ci.com
  2. Click the “Sign Up” button
  3. Choose “Sign up with GitHub”
  4. On the “Authorize Travis CI” page, click to “Authorize travis-pro”
  5. On the “First time here?” page, click the “Activate all repositories using GitHub Apps” button
  6. Select “All Repositories” (in general, you might want to be more selective in giving Travis-CI access to your repositories, but for now this is expedient). Click “Approve & install”
  7. On the Travis-CI page for your account, click “Activate” to activate the GitHub integration.
