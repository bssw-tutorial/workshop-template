Codecov Account

Used in lessons: Continuous Integration

Codecov facilitates visualizing and tracking of testng code coverage. It is easily integrated with hosting services, like GitHub and Travis CI.

You will need an account at codecov.io for the continuous integration testing activity. Basic Codecov accounts are free. When creating your account, you should link it to your GitHub account. Of course, this requires that you setup your GitHub account first.

Setting Up Your Codecov Account

  1. Go to https://codecov.io/
  2. Click the “Sign Up” button
  3. Choose “Sign up with GitHub”
  4. On the “Authorize Codecov” page, click to “Authorize codecov”
