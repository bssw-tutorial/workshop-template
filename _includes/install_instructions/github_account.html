GitHub Account

Used in lessons: Agile Methodologies, Git Workflows, Refactoring, Continuous Integration

You will need an account at github.com for many of the hands-on activiies. Basic GitHub accounts are free. We encourage you to create a GitHub account if you don't have one already. Please consider what personal information you'd like to reveal. For example, you may want to review these instructions for keeping your email address private provided at GitHub.

You will also need a supported web browser.

Setting Up Your GitHub Account

  1. Go to https://github.com
  2. Click the “Sign Up” button
  3. Complete the signup form
  4. Verify your account, as requested
  5. On the “Welcome to GitHub” page, scroll to the bottom and click on “Complete Setup”
  6. Complete the email verification process
  7. Login to GitHub with your new account
